# Job-Track Extension Icons

Place PNG icons here:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

For development, you can create simple placeholder icons using any image editor or online tool.

Example using ImageMagick:
```bash
convert -size 16x16 xc:#4cc9f0 icon16.png
convert -size 48x48 xc:#4cc9f0 icon48.png
convert -size 128x128 xc:#4cc9f0 icon128.png
```
